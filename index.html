HTML5 and ARIA

All the markup has semantic meaning. I have used all the relevant HTML5 semantic elements, along with WAI-ARIA roles for the best accessibility. No mark-up has been added simply for styling.

Read about creating simple, semantic markup.

Responsive

The design's simplicity means it easily responds to different screen sizes.

Simple designs are more responsive.

Modular

The style elements are modular, meaning that the markup or styling can be added into different pages and still work.
